Amazon Homepage Clone
This project is a basic clone of the Amazon homepage. It is built with HTML and CSS.
The purpose is to practice page layout, structure, and styling.

Project Overview:
The page includes a header, navigation bar, banner image, product boxes, and a footer.
There is no JavaScript and no backend.
It is a simple front-end practice project.

Features:-
• Header and navigation bar
• Banner image
• Product category boxes
• Footer section
• Organized folder structure for images and styles

Technologies Used:-
• HTML5
• CSS 3
